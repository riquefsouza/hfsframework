Parâmetros de Configuração
Categoria do Parâmetro Parâmetro Valor Descrição
Parâmetros do Tribunal NOME_TRIBUNAL Tribunal Regional do Trabalho da 1a. Região Nome do tribunal onde o sistema está instalado.
Parâmetros do Tribunal SIGLA_TRIBUNAL TRT1 Sigla do tribunal onde o sistema está instalado.
Parâmetros do Tribunal CODIGO_UNIDADE_GESTORA 080009 Código númérico de 6 dígitos que identifica o órgão no SIAFI.
Parâmetros do Tribunal APELIDO_UNIDADE_GESTORA 102 Código númérico de 3 dígitos da UG no código de barras da GRU.
Parâmetros de Login BLOQUEAR_LOGIN false Bloquear o sistema para que os usuários, exceto do administador, não façam login
Parâmetros de Login AMBIENTE_SISTEMA Produção Define o ambiente onde o sistema está instalado. Este parâmetro pode ser preenchido com: desenvolvimento, homologação ou produção
Parâmetros de Login ATRIBUTO_LOGIN NOME_USUARIO Define o atributo usado para efetuar login no sistema. Este parâmetro pode ser preenchido com: NOME_USUARIO ou CPF
Parâmetros de E-mail SMTP_SERVIDOR smtp.trt1.jus.br Servidor SMTP para que o sistema envie e-mail.
Parâmetros de E-mail SMTP_PORTA 25 Porta do servidor SMTP para que o sistema envie e-mail.
Parâmetros de E-mail SMTP_USERNAME Usuário para login no servidor SMTP.
Parâmetros de E-mail SMTP_PASSWORD Senha para login no servidor SMTP.
Parâmetros de E-mail SMTP_EMAIL_FROM sistema@trt1.jus.br E-mail do sistema.
Parâmetros de Conexão de Rede PROXY_SERVIDOR bravo.trtrio.gov.br Servidor do Proxy.
Parâmetros de Conexão de Rede PROXY_PORTA 8080 Porta do Proxy.
